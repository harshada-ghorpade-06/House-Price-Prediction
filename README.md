🏠 House Price Prediction
📌 Overview

This project predicts house prices using machine learning models.

⚙️ Steps
Data preprocessing (missing values, encoding, scaling)
Exploratory Data Analysis (EDA)
Model training (Linear Regression, Decision Tree, Random Forest, SVM)
Model evaluation (MAE, MSE, R² score)
🏆 Best Model
Random Forest performs best in most cases
🚀 Run
pip install -r requirements.txt
python main.py
📈 Output
Predicted house prices
